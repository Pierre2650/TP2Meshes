# Modélisation Géométrique - GAMAGORA
## Traitement des maillages
Projet Unity pour le traitement des fichier maillages OFF.

## Description
Le repertoire contient le code source d'un projet Unity dans lequel j’ai développé un script permettant de lire un fichier OFF, calculer ces normales et enlever un nombre de faces au maillage puis exporter ce meme fichier OFF.
